# 校园二手书流转交易平台 - 业务使用说明

## 一、系统概述

本平台是一个面向校园的二手书流转交易平台，支持学生发布、浏览、购买二手书籍，以及以书换书等功能。系统包含学生端和管理员端两个角色。

**技术栈：** Spring Boot 2.7 + MyBatis-Plus + Vue 2 + Element UI + MySQL 8

**访问地址：**
- 前端页面：http://localhost:8080
- 后端API：http://localhost:8088

## 三、学生端功能

### 3.1 注册与登录

**注册流程：**
1. 访问 http://localhost:8080，点击"立即注册"
2. 填写用户名、密码、手机号、昵称
3. 点击"注册"完成账号创建
4. 注册成功后自动跳转到登录页

**登录流程：**
1. 输入用户名和密码
2. 点击"登录"
3. 登录成功后进入首页

### 3.2 首页浏览与搜索

**浏览书籍：**
- 首页展示所有在售书籍，以卡片形式展示封面、书名、售价、原价、卖家信息
- 支持分页浏览

**搜索书籍：**
- 在搜索框输入关键词（书名、作者、ISBN）
- 可选择分类筛选
- 点击搜索按钮或按回车键执行搜索

### 3.3 发布书籍

**操作路径：** 顶部导航 → 发布书籍

**填写信息：**
1. 书名（必填）
2. 作者
3. 出版社
4. ISBN
5. 分类（必填）：计算机科学、数学、英语外语、文学小说、经济管理、理工教材、社会科学、艺术设计、考试考证、其他
6. 品相（必填）：全新、九成新、八成新、七成新及以下
7. 原价（必填）
8. 售价（必填）
9. 书籍描述
10. 书籍图片（最多5张，第一张为封面）

**编辑书籍：** 在"我的发布"中点击"编辑"按钮

### 3.4 书籍详情

**操作路径：** 点击任意书籍卡片进入详情页

**可执行操作：**
- **我想要：** 下单购买此书（创建订单）
- **收藏：** 收藏/取消收藏此书
- **联系卖家：** 跳转到聊天页面与卖家沟通
- **换书：** 选择自己的一本书发起换书请求

### 3.5 我的订单

**操作路径：** 右上角头像 → 我的订单

**买家视角（我买到的）：**
| 订单状态 | 可执行操作 |
|---------|-----------|
| 待付款 (PENDING) | 去支付、取消 |
| 已付款 (PAID) | 等待卖家发货 |
| 已发货 (SHIPPED) | 确认收货 |
| 已完成 (COMPLETED) | 评价、申请退款 |

**卖家视角（我卖出的）：**
| 订单状态 | 可执行操作 |
|---------|-----------|
| 已付款 (PAID) | 发货 |
| 申请退款 (REFUNDING) | 同意退款 |

**订单详情：** 点击"详情"可查看订单状态流转

### 3.6 退款流程

1. 买家在已完成订单中点击"申请退款"，填写退款原因
2. 卖家在"我卖出的"中看到退款申请，点击"同意退款"
3. 退款完成后订单状态变为 REFUNDED

### 3.7 评价系统

- 订单完成后，买家可对订单进行评价
- 评分：1-5星
- 文字评价
- 评价后自动更新卖家信誉评分

### 3.8 我的收藏

**操作路径：** 右上角头像 → 我的收藏

- 查看所有已收藏的书籍
- 点击书籍卡片跳转详情页
- 点击星标取消收藏

### 3.9 以书换书

**发起换书：**
1. 在书籍详情页点击"换书"
2. 选择自己要交换的书籍
3. 填写留言
4. 发送换书请求

**换书记录：** 右上角头像 → 交换记录

**换书状态流转：**
| 状态 | 说明 |
|------|------|
| 待处理 (PENDING) | 等待对方处理 |
| 已接受 (ACCEPTED) | 对方已接受，等待线下交易 |
| 已完成 (COMPLETED) | 双方确认换书完成 |
| 已拒绝 (REJECTED) | 对方拒绝了换书请求 |
| 已撤回 (CANCELLED) | 发起方撤回了请求 |

**操作说明：**
- 收到的请求：可接受或拒绝
- 已接受的请求：双方均可点击"确认完成"
- 发起的待处理请求：可撤回

### 3.10 即时通讯（聊天）

**操作路径：** 顶部导航 → 消息，或在书籍详情页点击"联系卖家"

**功能说明：**
- 左侧为会话列表，显示最近联系人
- 右侧为聊天窗口
- 支持实时消息收发（基于WebSocket）
- 未读消息数显示

### 3.11 个人中心

**操作路径：** 右上角头像 → 个人中心

**功能模块：**
- **个人信息：** 修改昵称、手机号、邮箱、头像
- **收货地址：** 添加/编辑/删除收货地址，设置默认地址
- **修改密码：** 修改登录密码
- **我的发布：** 查看已发布的书籍
- **我的订单：** 跳转到订单页面
- **我的收藏：** 跳转到收藏页面

## 五、订单状态流转

```
下单 → 待付款(PENDING) → 已付款(PAID) → 已发货(SHIPPED) → 已完成(COMPLETED)
         ↓                    ↓                                    ↓
      已取消(CANCELLED)   申请退款(REFUNDING) → 已退款(REFUNDED)   评价
                             ↓
                          争议(DISPUTE) → 管理员裁定
```
